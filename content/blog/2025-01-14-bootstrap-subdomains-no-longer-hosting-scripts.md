Salt Project Community Members!

`salt-bootstrap` scripts are now exclusively available for download via GitHub Releases. `winbootstrap.saltproject.io`
and `bootstrap.saltproject.io` will not be hosting the bootstrap scripts moving forward, but instead host a document
detailing how to download from the new locations.

The original notice was for scripts no longer being available after the **end of November 2024**, but we had extended this window.

Refer to [Salt Project Blog - Upcoming bootstrap subdomain decommission](/blog/upcoming-bootstrap-decommission) for more details.
