# Salt 3006.1 is available

The Salt Project has just released 3006.1 bugfix of Salt. To download and
install Salt 3006.1, see the
[Salt install guide](https://docs.saltproject.io/salt/install-guide/en/latest/).
To access the 3006.1 packages directly, go to the
[Salt Project Package Repository](https://repo.saltproject.io/salt/py3/).

- Release notes can be found here: https://docs.saltproject.io/en/latest/topics/releases/3006.1.html
- Changelogs can be found here: https://github.com/saltstack/salt/blob/v3006.1/CHANGELOG.md
- Sources are available on PyPI here: https://pypi.org/project/salt/3006.1/

Thank you all for your contributions!
