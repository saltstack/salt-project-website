# Salt 3005.1-4 Security Package Update

The Salt Project has released v3005.1-4 packages. This release only impacts the
onedir packages and the classic Windows and Mac packages. The package updates
include the following:

- Update gitpython to 3.1.30 (https://github.com/advisories/GHSA-hcpj-qp55-gfph)
- Update wheel to 0.38.4 (https://github.com/advisories/GHSA-qwmp-2cf2-g9g6)

| Note  |
| ---------------- |
| The Windows classic package version is 3005.1-5 for the new update. |

These are CVE fixes for the upstream projects listed above. We recommend you
update if you are using onedir packages or the classic Windows and Mac packages.
