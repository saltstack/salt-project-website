The Salt Project has just released 3007.1 STS bugfix of Salt. To download and
install Salt 3007.1 STS, see the
[Salt install guide](https://docs.saltproject.io/salt/install-guide/en/latest/).
To access the 3007.1 packages directly, go to the
[Salt Project Package Repository](https://repo.saltproject.io/salt/py3/).

- Release notes: https://docs.saltproject.io/en/latest/topics/releases/3007.1.html
- Changelogs: https://github.com/saltstack/salt/blob/3007.x/CHANGELOG.md
- Sources are available on PyPI: https://pypi.org/project/salt/3007.1/

Thank you all for your contributions!
