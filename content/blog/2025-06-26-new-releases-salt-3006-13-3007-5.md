Salt Project Community Members!

The Salt Project has just released the 3006.13 LTS and 3007.5 STS bugfixes of Salt. To download and
install, see the
[Salt install guide](https://docs.saltproject.io/salt/install-guide/en/latest/).

> **NOTE:** These releases include bugfixes to address problems with gitfs and master/minion connectivity issues experiences by users in the previous CVE releases.

**Salt 3006.13 LTS**

- Release notes: https://docs.saltproject.io/en/3006/topics/releases/3006.13.html
- Changelogs: https://github.com/saltstack/salt/blob/v3006.13/CHANGELOG.md
- Sources are available on PyPI: https://pypi.org/project/salt/3006.13/

**Salt 3007.5 STS**

- Release notes: https://docs.saltproject.io/en/3007/topics/releases/3007.5.html
- Changelogs: https://github.com/saltstack/salt/blob/v3007.5/CHANGELOG.md
- Sources are available on PyPI: https://pypi.org/project/salt/3007.5/

Directory repository locations:

- [Salt Project Repository: Linux (RPM)](https://packages.broadcom.com/artifactory/saltproject-rpm): Where Salt `rpm` packages are officially stored and distributed.
- [Salt Project Repository: Linux (DEB)](https://packages.broadcom.com/artifactory/saltproject-deb): Where Salt `deb` packages are officially stored and distributed.
- [Salt Project Repository: GENERIC](https://packages.broadcom.com/artifactory/saltproject-generic): Where Salt Windows, macOS, etc. (non-rpm, non-deb) packages are officially stored and distributed.

Thank you all for your contributions!

-- Salt Project Team
