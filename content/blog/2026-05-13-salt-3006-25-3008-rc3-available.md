# Salt Project Announcement: New LTS & RC Releases Available

The Salt Project is excited to announce the release of **Salt 3006.25 LTS** (Bugfix Release) and **Salt 3008 RC3** (Release Candidate).

Whether you are prioritizing long-term stability or looking to test the latest upcoming features, we have new builds ready for your environment.

## Salt 3006.25 LTS (Bugfix Release)

This release provides critical bugfixes and stability improvements for the 3006 Long-Term Support branch. We recommend all users on the 3006.x series upgrade to this version.

* **Install Guide:** [Salt Install Guide](https://docs.saltproject.io/salt/install-guide/en/latest/)
* **Release Notes:** [3006.25 Release Notes](https://docs.saltproject.io/en/3006/topics/releases/3006.25.html)
* **Changelog:** [GitHub Changelog](https://github.com/saltstack/salt/blob/v3006.25/CHANGELOG.md)
* **Source:** [PyPI - Salt 3006.25](https://pypi.org/project/salt/3006.25/)

## Salt 3008 RC3 (Release Candidate)

We are moving closer to the final release of **Salt 3008 LTS**. We invite the community to download and test Release Candidate 3 (RC3) to help us ensure the highest quality for the upcoming LTS.

* **How to Test:** [Install a Release Candidate](https://docs.saltproject.io/salt/install-guide/en/latest/topics/other-install-types/release-candidate.html)
* **The Tag:** [v3008.0rc3 on GitHub](https://github.com/saltstack/salt/tree/v3008.0rc3)
* **PyPI Source:** [v3008.0rc3 on PyPI](https://pypi.org/project/salt/3008.0rc3/)

## Installation & Upgrade Notes

Users can install Salt via **PyPI** or our **packages**

> [!IMPORTANT]
> We support differing versions between Master and Minions, but as always, it is **highly recommended to upgrade your Master first.**

## Reporting Issues

If you encounter any issues with the software or the packaging, please let us know by opening an issue on our GitHub repository:

[**Open a GitHub Issue**](https://github.com/saltstack/salt/issues/new/choose)

Thank you for being a part of the Salt Project community!
