Salt Project Community Members!

In continued efforts to improve automation, docs publishing will experience an update on 2024-12-18. Downtime is expected to be minimal as the Salt Project documentation subdomain (`docs.saltproject.io`) will be maintained by a new GitHub repository, without any disruption or changes to DNS.

Migrating the domain between different GitHub repositories will be done around 10AM MST.
