Salt Project Community Members!

A new version of `salt-bootstrap` has been released, **which includes more updates making use of `packages.broadcom.com`**:

- Photon OS now supports minor and major versions of Salt, for example: `3006.8`, or `3006` which will get `3006.9`
- Linux platforms now support minor versions, for example: `3006.7`, or `3006.9` as the latest for `3006`

See details on [Salt Project Blog - salt-bootstrap Breakage and Next Steps: New Updates](/blog/salt-bootstrap-breakage-part-2) for where to download the latest `salt-bootstrap` scripts.
