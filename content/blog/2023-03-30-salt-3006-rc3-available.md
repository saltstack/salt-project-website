# Salt 3006 RC2 is now available

The Salt Project has just released RC3 (release candidate 3) of the Salt 3006
LTS. To download and test Salt 3006 RC3, see
[Install a release candidate](https://docs.saltproject.io/salt/install-guide/en/latest/topics/release-candidate.html)
in the Salt install guide.

Users can install via PyPI, packages, or use the Docker containers.

| Note  |
| ----- |
| We support differing versions between master and minions, but as always it is recommended to upgrade your master first. |

- The tag: https://github.com/saltstack/salt/tree/v3006.0rc3
- The PyPI Source: https://pypi.org/project/salt/3006.0rc3/
- Docker Containers: https://hub.docker.com/r/saltstack/salt

If you find any issues with Salt or the packaging, open an issue here:
[https://github.com/saltstack/salt/issues/new/choose](https://github.com/saltstack/salt/issues/new/choose)
