# The New Mojo Programming Language.

Tom loves learning new programming languages, always has. He even likes learning about the ones that he thinks are--in a word--crap!  Even though he has this obsession, he's always maintained that his favorite is Python.  

Recently, Tom came across a brand-new programming language called "Mojo"!  Like Tom, it's safe to say that we're all fans of Python, generally speaking. However, Python is not without its faults.  That’s where Mojo comes in. The Mojo development team is present it to the world as kind of a "Python + +".  Simply put, it addresses Pythons biggest problem, which is speed and takes actions to make it fast... Ok, ok, faster. It also claims to make a number of additional improvements, as well.

In this episode of The Hacks, Tom and Chunga everything they know so far about Mojo and they'll throw in a few other new programming languages for good measure.  Listen NOW!
