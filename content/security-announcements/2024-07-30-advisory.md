The Salt Project has just released 3006.9 LTS bugfix of Salt.

> **NOTE**: This bugfix release includes at least one CVE fix (low to medium risk). See release notes below for details.

To download and install Salt 3006.9 LTS, see the
[Salt install guide](https://docs.saltproject.io/salt/install-guide/en/3006/).
To access the 3006.9 packages directly, go to the
[Salt Project Package Repository](https://repo.saltproject.io/salt/py3/).

- Release notes: https://docs.saltproject.io/en/3006/topics/releases/3006.9.html
- Changelogs: https://github.com/saltstack/salt/blob/3006.x/CHANGELOG.md
- Sources are available on PyPI: https://pypi.org/project/salt/3006.9/
