# Salt Formulas

Join the Salt Formulas working group to help create best practices and workflows
for using Salt in real deployments.

Formulas are pre-written Salt States. They are as open-ended as Salt States
themselves and can be used for tasks such as:

- Installing a package
- Configuring and starting a service
- Setting up users or permissions
- Many other common tasks
