## Salt Project Website

The main static site for https://saltproject.io, built with [Hugo](https://gohugo.io/)!

This website currently uses the [Universal Hugo Theme](https://github.com/devcows/hugo-universal-theme), and is mostly made up of templated data found in the `exampleSite` location.

### Prereqs

- [Install Hugo](https://gohugo.io/installation/)
  - Version `v0.133.0` or higher
  - This was built with the `extended` version of Hugo, but may not be required
- Git

### Build Locally

```
# Requires Universal theme
git clone git@github.com:devcows/hugo-universal-theme.git themes/hugo-universal-theme

# Serves for viewing changes locally
# Dynamically loads when changes happen in repo
hugo serve
```
